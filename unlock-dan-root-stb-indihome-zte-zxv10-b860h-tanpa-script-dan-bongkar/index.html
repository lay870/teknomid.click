Tutorial Unlock Dan Root STB  ZTE ZXV10 B860H

Tutorial Unlock Dan Root STB Indihome ZTE B860H – Peringatan! Tulisan ini hanya untuk bahan pembelajaran untuk memanfaatkan STB bekas Indihome saja, resiko akibat dari penerapan tutorial ini adalah tanggung jawab kamu sendiri.

Bagi pelanggan indihome dengan paket internet dan TV pasti sudah tidak asing dengan tambahan perangkat Set Top Box (STB) Android yang bisa kamu hubungkan langsung ketelevisi atau monitor untuk menyaksikan siaran televisi secara langsung, on demand, maupun delay.

STB ini menggunakan sistem operasi android, namun kita tidak bisa dengan leluasa menginstall aplikasi android yang kita inginkan karna perangkat ini dikunci oleh pihak telkom.

Terkadang ketika kita sudah berhenti berlangganan indihome, perangkat STB ini tidak diambil lagi oleh pihak telkom.

Yang mengakibatkan perangkat ini tidak berguna karena tidak dapat digunakan untuk menonton TV maupun menginstall aplikasi lain, seperti youtube untuk melakukan streaming video.

Untuk itu kita perlu melakukan unlock pada perangkat ini agar dapat digunakan untuk masuk kedalam sistem dan menginstall aplikasi android yang kita inginkan.

Sebelum memulai tutorial ini ada beberapa hal yang wajib kamu persiapkan:

  • Laptop atau PC Desktop dengan OS Windows
  • Kabel USB Male to Male
  • STB Bekas Indihome ZTE ZXV10 B860H

Cara Unlock Dan Root STB ZTE B860H

1. Install USB Burning Tools

Install aplikasi USB Burning Tools ini pada Laptop atau PC Desktop bisa kamu unduh disini

2. Install Amlogic Driver

Download disini, ekstrak file tersebut dimana saja yg kamu inginkan. Lalu buka Device Manager – masuk tab action – add legacy hardware – install manual (advance) – next – next have a disk dan pilih browse cari file WorldCup_Device lalu ok.

3. Pulpstone Amlogic Update USB Tool

Bisa kamu unduh Disini, file ini sudah termasuk file system lengkap, setelah selesai mengunduh kamu ekstrak file tersebut dimana saja pada PC.

Baca Juga: Tutorial Unlock STB ZTE B860H Dengan Pulpstone

4. Lakukan Factory Reset

Sebelum memulai proses unlock dan root, silahkan lakukan factory reset pada perangkat STB B860H kamu dengan masuk ke menu setting (pengaturan) – about (tentang) – data factory reset – lalu masukan password 6321 – OK

5. Buka aplikasi USB Burning Tools

Silahkan buka aplikasi tersebut dan ubah kebahasa inggris, aplikasi ini digunakan hanya untuk mendeteksi apakah STB 860h kamu sudah terdeteksi pada komputer.

6. Koneksikan STB dengan Laptop/PC

Cabut kabel power dari STB. Tahan tombol power lalu hubungkan USB Male To Male pada slot USB ke-2 pada STB dan Laptop/PC. Periksa aplikasi USB burning tools, apabila sudah muncul tulisan connect berarti STB kamu sudah terbaca oleh sistem.

7. Buka Aplikasi Pulpstone Amlogic Update USB Tool

Buka aplikasi tersebut dari folder yang telah kamu ekstrak sebelumnya. Tekan 1 – enter – 2 – enter dan tunggu hingga proses selesai sampai 100%. Pastikan semua command tersebut berakhir dengan muncul tulisan succes. Selesai proses instalasi dan kamu bisa bisa mencabut STB lalu hubungkan pada televisi.

8. Wajib Dilakukan Setelah Instalasi

Setelah instalasi selesai hubungkan ke televisi kamu, lalu ketika STB sudah menyala Set TVLauncher – Always, lakukan Factory Data Reset [WAJIB] dengan menuju ke “Settings – More Settings – Backup & Reset”.
