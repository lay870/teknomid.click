    Aplikasi Design Open Source di Linux dan Windows

    Teknomid - Sekarang teman-teman tidak perlu lagi melakukan pembajakan atau membeli aplikasi berbayar dengan harga yang mahal sebagai designer pemula. Bukan hanya itu bagi kalian pengguna linux maupun windows yang masih sering me-reboot mesin tempurnya hanya untuk mengakses aplikasi – aplikasi dari Adobe sebaiknya mulai mencoba menggunakan  Aplikasi Design di Linux yang Gratis dan Open Source berikut ini.

    1. Inkscape

    Inkscape merupakan sebuah perangkat lunak design pengolah gambar vektor di bawah lisensi GNU GPL yang bersifat free, open source, dan tentunya gratis untuk di gunakan tanpa biaya sepeserpun.

    Ia dapat berjalan pada berbagai macam platform serta menggunakan format utama SVG untuk filenya dengan mengacu kepada standard dari W3C. Untuk saat ini pengguna Inkscape sudah sangat banyak sekali baik itu dari kalangan profesional maupun pengguna umum untuk kebutuhan design.

    Ketika tulisan ini di buat Inkscape sudah mencapai rilisan dengan versi 1.1

    Secara umum Inkscape sudah tersedia hampir di sebagian besar repositori dari berbagai macam Distro Linux dan mempunyai banyak komunitas juga tutorial yang tersebar di internet.

    Pada Ubuntu kalian tinggal menginstalnya melalui Ubuntu Software Center atau melalui shell dengan mengetikan perintah "sudo apt install inkscape"

    Atau bisa juga download applikasi standalone langsung dari website inkscape file-size: 110 MB dan lakukan perintah berikut lewat terminal.

    chmod u+x Inkscape-3bf5ae0-x86_64.AppImage
    sudo chown teknomid:teknomid Inkscape-3bf5ae0-x86_64.AppImage
    sudo cp Inkscape-3bf5ae0-x86_64.AppImage /bin/inkscape
    -
    Masukkan Perintah di terminal : inkscape

    Sumber: https://inkscape.org/

    2. Krita

    Design Krita adalah sebuah pengolah gambar yang tidak kalah keren dengan Inkscape, namun aplikasi satu ini merupakan aplikasi pengolah gambar raster yang lebih berfokus kepada painting dan animasi.

    Sama halnya dengan Inkscape, Krita juga bersifat free dan open source tentunya Krita juga bebas di gunakan tanpa di pungut biaya sepeserpun alias gratis. Krita sendiri mengusung Design antarmuka yang elegant dan mirip dengan aplikasi Illustrator dari Adobe.
    Versi terbaru sekarang ini sudah update ke version Krita v4.4.8

    Sebagai salah satu pengolah gambar raster yang lebih fokus ke penggambaran daripada fungsi melakukan pengeditan Krita juga mempunyai nilai plus yaitu bisa di gunakan untuk membuat dan mengolah gambar vektor seperti halnya Illustrator.

    Krita juga dapat berjalan di berbagai macam platform. Untuk menginstall Krita pada Ubuntu cukup melalui Ubuntu Software Center atau dengan menggunakan perintah sudo apt install krita dan Krita siap di gunakan.

    Atau bisa juga download applikasi standalone langsung dari website krita file-size: 225 MB dan lakukan perintah berikut lewat terminal.

    sudo chown teknomid:teknomid krita-4.4.8-x86_64.appimage
    sudo cp krita-4.4.8-x86_64.appimage /bin/krita
    -
    Masukkan Perintah di terminal : krita

    Sumber: https://krita.org/
