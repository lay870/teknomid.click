Forward Domain Lama Ke Domain Baru Mudah Pakai Cloudflare

Forward Domain Lama Ke Domain Baru Menggunakan Page Rule Cloudflare – Kalau kamu berniat melakukan re-branding nama domain, maka kamu harus melakukan Wildcard Domain Forwarding, proses ini melakukan redirect nama domain lama situs ke nama domain baru.

Kamu tidak perlu menggunakan cara lama pada hosting untuk melakukan forwarding domain, cukup menggunakan fitur page rule yang ada pada Cloudflare.

Cloudflare memiliki 203 cloud server di seluruh dunia, maka dari itu penggunaan Page Rule pada Cloudflare dapat mempercepat proses forwarding domain baru kamu. Berikut langkah-langkah yang harus kemu lakukan.

Contohnya, kamu ingin mengganti domain yang asal mula teknomid.net menjadi teknomid.com.

Cara Forward Nama Domain Lama Ke Domain Baru Dengan Cloudflare

Masuk Cloudflare

Buat akun Cloudflare atau masuk jika kamu sudah memiliki akun.

Tambahkan Situs

Pastikan nama domain lama sudah ada di Cloudflare dan masuk ke pengaturan domain lama.

Tambahkan Page Rule

Untuk melakukan redirect domain lama ke domain baru, masuk ke menu Page Rule pada Cloudflare dan lakukan pengaturan seperti gambar.
