Cara Menghemat Kuota, Begini Cara Mematikan Download Otomatis di WhatsApp

WhatsApp menjadi aplikasi paling banyak memiliki pengguna aktif hingga sekarang ini. Berkat layanan dan keamanan privasinya, membuat aplikasi WhatsApp semakin banyak digemari oleh pengguna. Hal tak kalah penting adalah soal enkripsi end-to-end yang dapat membuat percakapan pengguna tidak bisa diintip oleh orang lain bahkan termasuk pihak WhatsApp sendiri.

Namun, tahukah Anda jika WhatsApp di dalamnya memiliki fitur yang namanya Unduhan Otomatis terhadap media-nya. Fitur ini akan secara otomatis mengunduh media yang masuk ke WhatsApp Anda, baik melalui percakapan pribadi ataupun melalui Grup WhatsApp. Media yang secara otomatis akan terunduh jika Anda mengaktifkan fitur ini adalah berupa gambar, video, file dan lainnya.

Fitur unduhan otomatis atau yang disebut dengan Auto Download pada aplikasi WhatsApp berfungsi untuk menyimpan foto atau video supaya bisa langsung tersimpan pada galeri perangkat Anda. Sekilas hal ini memang baik dan membuat Anda tidak kehilangan file media yang masuk ke WhatsApp Anda melalui percakapan. Namun tahukah jika Unduhan Otomatis ini akan menguras Kuota Internet Anda?

Namun tidak perlu khawatir, fitur tersebut dapat Anda nonaktifkan untuk menghindari kehabisa kuota ataupun penyimpanan Anda menjadi penuh. Nah, cara agar file media WhatsApp tersebut tidak tersimpan pada perangkat Anda cukup mudah dan Anda bisa bisa mengaturnya sendiri. Untuk itu, berikut ini Penulis berikan langkah-langkah cara mudah mematikan fitur Unduhan Otomatis pada WhatsApp.

Begini Cara Mematikan Unduhan Otomatis di WhatsApp

  1. Pertama, silahkan buka aplikasi WhatsApp yang sudah terinstal di ponsel Anda. Tap atau ketuk ikon titik tiga pada pojok kanan atas, Lalu pilih menu Pengaturan atau setting.
  2. Klik opsi Data dan Penyimpanan atau Storage and Data. Nantinya Anda akan melihat opsi Unduh Otomatis Media atau Media auto-download.
  3. Pada bagian bawahnya ada tiga opsi yang bisa Anda pilih, yaitu When using mobile data, When connected on Wifi, atau When roaming.
  4. Nah, supaya semua media atau file tidak tersimpan secara otomatis pada galeri, Anda perlu menghilangkan centangnya. Hapus semua centang pada opsi foto, audio, video dan dokumen. Lakukan hal sama pada ketiga opsi tersebut, When using mobile data, When connected on Wi-Fi, atau When roaming.
  5. Jika sudah, klik Ok untuk konfirmasi dan selesai. Jangan lupa pastikan ketiga keterangannya sudah berubah menjadi No Media.

Sekarang, setelah Anda mematikan fitur unduh otomatis, maka semua jenis media file yang masuk ke WhatsApp bisa Anda sortir sendiri mana yang akan Anda buka atau tidak. Jika Anda membukanya, berarti Anda secara otomatis akan menyimpan pada Galeri ponsel Anda. Semoga bisa bermanfaat untuk Anda.
